# qmk
QMK keyboards

Small repo with custom whateverany keyboard code, maps and tools.

To compile, symlink keyboards to qmk_firmware/keyboards. e.g.

```
cd qmk_firmware/keyboards
ln -s ../../qmk/keyboards/whateverany
```
